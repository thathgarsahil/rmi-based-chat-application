GUI Based Chat Application
Project by - Sahil

1. Go to Directory upto 'cmd' by using cd command.
2. Compile -
	javac -d . Server/ChatServerMain.java
	javac -d . Client/ChatFrame.java
3 Run in separate command prompt windows - 
	java -cp . Server/ChatServerMain JavaChat &
	java -cp . Client/ChatFrame Sahil localhost JavaChat &

			OR

1.  In Netbeans, run ChatServerMain.
		- JavaChat
2. Then run ChatFrame.
		- User
		- localhost
		- JavaChat
